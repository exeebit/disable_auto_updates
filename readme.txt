A simple plugin to disable themes, core and plugin auto updates

== Description ==

This is a very simple plugin through which you can easily disable all kind of updates of your wordpress site. All you have to do is to choose what to disable. You can later enbale it easily if you need in any case. You can also hide update notifications also through this plugin. Number of updates appear with plugin (red colored) will disappear after you disable update notification.

You can disable just plugin, theme or core update. You will also see a console where who has changed what will be logged along with your wordpress time. You can also disable irritating with just one click. 

Check our other project. [View](https://exeebit.com/wordpress)

== Updates ==

== Installation ==

You can [download](https://downloads.wordpress.org/plugin/disable-auto-updates.zip) and upload the plugin via Admin > Plugins > Add New > Upload Plugin. Or,
Go to your website's admin panel. Select Plugins > Add New. search for **Disable Auto updates** by [Exeebit](https://exeebit.com). Click **Install Now button**. Then simply click **Active**.
